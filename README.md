# Tic-Tac-Toe 🎮

A classic Tic-Tac-Toe game built with React and Vite. 

This repository also serves as a personal sandbox for mastering Git and GitHub workflows, including branching, pull requests, issue tracking, and resolving merge conflicts in a simulated team environment.

## 🛠 Technologies Used
* **Frontend:** React (JSX), CSS
* **Build Tool:** Vite
* **Version Control:** Git, GitHub, GitHub Desktop
* **Editor:** VS Code

## 🚀 How to Run Locally

If you want to clone and run this project on your local machine, follow these steps:

**1. Clone the repository**
```bash
git clone [https://github.com/deividmomchilov/tic-tac-toe.git](https://github.com/deividmomchilov/tic-tac-toe.git)
cd tic-tac-toe
npm install
npm run dev
